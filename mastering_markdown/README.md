# Guia de Sintaxe

## Headers
```
# isso é uma <h1> tag
## isso é uma <h2> tag
###### isso é uma <h6> tag
```


## Ênfase
*Texto em itálico*

_Outro texto em itálico_

**Texto em negrito**

__Outro texto em negrito__

_Você **pode** combina-los_

## Listas

### Não ordenadas
* Item 1
* Item 2
  * Item 2a
  * Item 2b
  
### Ordenadas
1. Item 1
1. Item 2
1. Item 3
   1. Item 3a
   1. Item 3b
